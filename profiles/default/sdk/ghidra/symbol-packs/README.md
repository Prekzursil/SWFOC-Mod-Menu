# Ghidra Symbol Packs

`SignatureResolver` checks this folder for `<fingerprintId>.json` packs before running pattern scans.

## Contract

- File name: `<fingerprintId>.json`
- Schema: `tools/schemas/ghidra-symbol-pack.schema.json`
- Source: generated by `tools/ghidra/run-headless.ps1` or `tools/ghidra/run-headless.sh`

## Override

Set `SWFOC_GHIDRA_SYMBOL_PACK_ROOT` to use a different folder at runtime.
